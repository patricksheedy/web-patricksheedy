About This Calculator

This calculator demonstrates vanilla JavaScript event handling, DOM manipulation, and mathematical operations. It features:

  • Decimal number support
  • Clear and clear entry functions
  • Keyboard input support
  • Responsive design

Technologies used: HTML5, CSS3, Vanilla JavaScript, Bootstrap 5
